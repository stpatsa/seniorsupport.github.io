Local Tax Services

1.Security: To keep yourself safe from fraudulent web page, always verify if the website has a small padlock on the search web bar. This means that the website is safe. If it has an exclamation point, the site is not safe. If you find a untrustworthy website, never give out info that is sensitive, like a login password, verification codes, or even your name and contact information. When in doubt, don’t fill it out. If you find a fradulent website, please report it to Google Safe Browsing (link is down below)

2.Check the URL: Check if the URL is misspelled or grammatical errors on the site (ex. amaz0n.com instead of amazon.com).

3.Run the page: Run the site through a website checker to verify if the website is secure. A secure website check can let you know if the site has vulnerabilities and if it’s using encryption.
