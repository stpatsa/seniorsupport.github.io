Here's Some Steps to Follow to Make Sure You're Reading Credible News!
If you want to avoid fake news sources, use this link in the bottom.

1.Check if the URL is misspelled or grammatical errors on the site (ex. amaz0n.com instead of amazon.com).


2.Look for the padlock on the website (pictured above). The padlock on the website shows that the site is secure. You can click on the lock to detect more information about the site. .

3.Run the site through a website checker to verify if the website is secure. A secure website check can let you know if the site has vulnerabilities and if it’s using encryption.
